# Secret of the Pocket (Tayna Karmana)

**Secret of the Pocket** is a mobile app designed to help users improve their financial literacy by exploring the story of financial institutions and trade in Nizny Novgorod.


## features
- **Interactive Cards**: explore architectural objects related to the city's financial activity - fair buildings, stock exchanges, warehouses, and more
- **Narrative Story**: an engaging story with characters through which the user learns financial history and historical events
- **Map**: a route with historical locations that can be completed in one walk or spread over several days
- **Audio Mode**: option to listen to the story instead of reading, hence users can read or listen to the story while walking or from home on a smartphone
- **Accessibility**: visually impaired mode with adjustable text size, color schemes, and option to hide images


## installation and running
1. make sure [flutter sdk](https://flutter.dev/docs/get-started/install) is installed
2. clone the repository
```bash
git clone https://github.com/sofa797/nn_finance_app.git
cd nn_finance_app
```
3. get dependencies
```bash
flutter pub get
```
4. run the app on your device or emulator
```bash
flutter run
```
